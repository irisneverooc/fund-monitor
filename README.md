# fund-monitor

一个面向个人投资学习的 Python 项目：记录基金持仓，计算每日收益，并生成**个人基金监控日报**（规则化日报）。

## 重要说明

- 当前 `data/funds.csv` 与净值数据均为模拟示例，仅用于功能演示，不代表真实持仓。
- 当前项目输出仅用于学习与记录，不构成任何投资建议。

## 项目目标

这个项目计划逐步实现：
- 基金持仓监控
- 净值变动跟踪
- Markdown 日报生成
- 更多投资信息整合与总结

第一阶段（当前）目标是搭建“基金监控日报框架”，先完成可运行的最小闭环，便于学习和后续扩展。

## 项目亮点

- 使用 pandas 进行数据读取与计算（环境缺少 pandas 时自动回退内置 csv 读取）
- 自动生成 Markdown 规则化基金监控日报
- 支持持仓收益、收益率、持仓占比、资产类型占比分析
- 使用规则引擎生成“今日观察”“交易观察提醒”“组合建议”“风险提示”
- 模块化结构清晰：**数据读取、收益计算、报告生成相互独立**，便于后续替换真实接口或扩展可视化

## 当前功能（MVP）

- 使用 CSV 保存基金池数据（含基金类型）
- 可通过修改 `data/funds.csv` 自定义个人基金池
- 支持 `units = 0` 的观察基金（日报中显示“观察基金”）
- 使用模拟净值数据（不调用真实接口）
- 计算每只基金的：
  - 当前市值
  - 持仓收益
  - 收益率
  - 持仓占比
- 生成一份 Markdown 日报到 `reports/` 目录
- 支持个基观察、交易观察提醒、组合建议、配置结构分析

## 项目结构

```text
fund-monitor/
├─ data/
│  └─ funds.csv                # 个人基金池配置（当前为模拟示例）
├─ reports/                    # 生成的日报文件
├─ src/

│  ├─ models.py                # 数据模型
│  ├─ data_loader.py           # 数据读取
│  ├─ mock_nav_provider.py     # 模拟净值提供器
│  ├─ calculator.py            # 收益计算与配置分析

```

## 如何运行

1. 进入项目目录：

```bash
cd fund-monitor
```

2. 运行脚本：

```bash
python src/main.py
```

3. 运行完成后，在 `reports/` 下查看日报文件。

## CSV 字段说明

`data/funds.csv` 示例：

```csv
code,name,fund_type,units,cost_nav
513100,纳斯达克100指数(QDII)ETF,海外指数,0,1.0000
161125,标普500指数(LOF)A(QDII),海外指数,0,1.0000
518880,黄金ETF,黄金,1200,4.2500
000216,华安黄金易(ETF联接)A,黄金,800,1.1800
```

字段含义：
- `code`：基金代码
- `name`：基金名称
- `fund_type`：基金类型（如海外指数、黄金、宽基指数）
- `units`：持有份额（`0` 表示观察基金）
- `cost_nav`：成本净值（买入均价）

说明：
- 当前 CSV 内容为个人基金池示例，可按自己的关注方向修改。
- 当前净值仍由 `src/mock_nav_provider.py` 提供模拟数据。
- 后续将接入真实基金净值接口并合并到日报。
- 当前交易观察提醒基于用户自定义规则，仅用于辅助复盘，不构成投资建议。
